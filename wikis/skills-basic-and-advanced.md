S kills are divided into two  categories: Basic  and Advanced.  Anyone  can  attempt  to  use  Basic  Skills, even if he doesn't have them trained (though training certainly increases his chances for success). Only characters with  specific  training  may  use  Advanced  Skills.  For  more information on making Skill Tests, see Chapter Ix: Playing the Game .

## Basic Skills

Basic Skills are those common to all people throughout the galaxy. Basic Skills are so rudimentary that a character can attempt to use one even if he doesn't have the skill trained. When  a  character  makes  a  Skill  Test  with  an  untrained Basic  Skill,  he  uses  half  the  value  of  the  skill's  governing Characteristic  (round  down).  Conversely,  if  a  character  has a  Basic  Skill  trained,  he  uses  the  full  value  of  the  skill's governing Characteristic.

## Advanced Skills

Advanced Skills require very specific training or education, and  relatively  few  individuals  in  the  galaxy  even  have them.  Unlike  Basic  Skills,  Advanced  Skills  cannot  be  used untrained.  Regardless  of  the  Explorer's  intrinsic  abilities and [Characteristics](starship-anatomy-detailed.md), for [Example](rules-tests.md), he would never be able to understand [The Eldar](faction-eldar-overview.md) Language without the benefit of formal instruction.

## Treating Advanced Skills as Basic Skills

Sometimes  an  Explorer  has  the  opportunity  to  treat  an Advanced Skill  as  a  Basic  Skill-usually  because  of  a  trait acquired on the character's Origin Path. This simply means that the Explorer can make Skill Tests with that skill, even if he doesn't have the skill trained. Such Skill Tests are made at half of the skill's governing Characteristic (round down), just as if it were a Basic Skill.
