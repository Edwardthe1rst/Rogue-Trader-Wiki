## Crafting Skills

Skills  with  the  Crafting  descriptor  allow  the  character  to create things from parts or raw materials. To create an item, a character must have the necessary materials, [Tools](equipment-tools.md), workspace, time, and skill-as determined by the GM. The character then makes the appropriate Skill Test as an Extended Action. On a success, the item is generated. The quality/finish/usefulness of the item improves with each degree of success. Failing by three degrees indicates some of the materials have been ruined in the process. Failing by five degrees or more indicates all the materials have been ruined.

## Exploration Skills

Skills with the Exploration descriptor can be used together in [Exploration Challenges](rules-exploration.md). These tasks are often lengthy and difficult  and  thus  work  best  as  group  efforts  with  several Explorers each contributing one or more areas of speciality. See page 263 for more information on [Exploration Challenges](rules-exploration.md).

## Interaction Skills

All skills with the [Interaction](rules-interaction.md) descriptor involve interplay with others. As a result, Interaction Skills usually involve [Opposed Tests](rules-tests.md)  (see  page  293).  The  difficulty  of  these  Skill  Tests depends on the individual's disposition towards the Explorer, as defined by the GM in accordance with the circumstances of  the  encounter.  See Chapter X: the Game Master for more information on NPC disposition.

## Investigation Skills

Most of the skills with the Investigation descriptor allow the Explorer to collect information from various sources, and also require [Extended Tests](rules-tests.md) as the Explorer hunts for clues in the ship's archives or in a [Rival](talents-descriptions.md)'s cogitator. The GM will set up the proper time period for the Tests using the material on page 264 as a guide, as it takes far longer to piece together the writings of a xeno philosopher than it does to interrogate the lackey of a local crime lord.

## Movement Skills

Skills with the Movement  descriptor  all relate to the Explorer's mobility, anything from daring leaps across chasms to [Climbing](combat-movement.md) out of the same if he doesn't quite make it across. They  involve  the  ability  to  move  across,  through,  over,  or between the Explorer's environment.

## Operator Skills

If a skill involves transportation, be it a humble ground vehicle or a mighty starship, it receives the Operator descriptor. These skills allow the Explorer to drive or pilot a variety of vehicles.
