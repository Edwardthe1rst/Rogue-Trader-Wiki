R ogue  TRadeR is  a  roleplaying  game  that  uses  tensided  polyhedral  dice  for  its  core  game  mechanic. These  dice  can  be  purchased  from  any  game  store, many  bookstores,  and  are  probably  available  at  the  same place this book was purchased. Each player and the GM will need access to at least two ten-sided dice, preferably of two different colours.
